## 1. ご提案の目的

客席のセルフオーダー端末 **16台** と、その運用に必要な周辺機器（ケース・スタンド・充電・Wi-Fi）について、**「1台あたり本体2万円前後」を予算の目安**としてご提案します。最終的な機器のご購入はオーナー様の判断となりますので、選んでいただくための比較材料としてご活用ください。

### 前提条件
- 客席に16台（カウンター席は電源つなぎっぱなしの常時起動、個室はスタッフが手渡し）
- **16台を確実に・同じ機種でまとめて調達できること**（故障時の差し替えや管理が楽になるため）
- 明るい店内でも見やすい画面（メニュー写真・動画・5言語表示）
- 注文画面だけを表示し、他の操作はできないように固定（キオスク化）
- 飲食環境のため、ケースは**拭ける・落としても割れにくい**ものを必須とする

> 機種選定では「価格」と「確実に16台揃うか（調達のしやすさ）」を最優先しました。スペックが高くても、安く買える時期が限られて16台分を揃えにくい機種は本命から外しています。

## 2. 3プラン比較（16台＋周辺機器の総額目安）

| | **プランA：本命（推奨）** | **プランB：予算内・高画質** | **プランC：品質・保守重視** |
| :--- | :--- | :--- | :--- |
| タブレット本体 | **アイリスオーヤマ LUCA TE10D1M64** | Xiaomi Redmi Pad SE 11" | Lenovo Tab B11 / M11 |
| 本体価格（1台） | **約1.7〜2.2万円** | 約1.9万円〜 | 約2.8〜3.8万円 |
| 16台まとめ調達 | **◎ 国内量販・法人窓口で確実** | △ 在庫変動あり・要確認 | ◎ 直販・法人で安定 |
| 画面の精細さ | 標準（1280×800） | **高精細（1920×1200）** | **高精細（1920×1200）** |
| 強み | 価格が安く、確実に16台揃う | 同じ予算で画面がきれい | 調達・保守が最も安定、画面も高品質 |
| 注意点 | 画面の精細さは標準（実機で要確認） | 16台分の在庫確保が前提 | 予算2万円を超える |
| **総額目安（16台＋周辺一式）** | **約60〜75万円** | **約60〜75万円** | **約80〜95万円** |

### 推奨プラン：A（アイリスオーヤマ LUCA TE10D1M64）
ご予算「1台2万円前後」に収まり、**国内の量販店・法人窓口で16台を確実にまとめて調達できる**点が最大の理由です。画面の精細さは標準的（1280×800）ですが、客席で注文する用途（写真・動画・メニュー表示）では十分実用的と見込んでいます。これは**まず1台を先行購入して実機で見え方を確認**してから16台へ進めば、リスクなく判断できます。

- **同じ予算で画面のきれいさを優先されるなら → プランB（Redmi Pad SE）**。ただし16台分の在庫がそろうか事前確認が必要です。
- **ご予算に余裕があり、調達・保守の安心と画面品質を最重視されるなら → プランC（Lenovo）**。1台あたり1万円前後高くなります（16台で約15〜25万円差）。

> ※高スペック機（Samsung Galaxy Tab A9+ 等）は、安く買える時期が限られ16台分をその価格で揃えにくいため、本提案では本命から外しています（ご希望あれば品質保険の選択肢としてご説明します）。

## 3. ケース・スタンド（用途で2種類に分けるご提案）

飲食環境では汚れが縫い目に溜まる**手帳型（合皮）カバーは不向き**のため、**拭ける素材**に統一します。本体機種が変わってもそのまま使えるものを選んでいます。

| 設置場所 | おすすめ | 理由 | 価格目安（1台） |
| :--- | :--- | :--- | :--- |
| **カウンター席（据え置き）** | 薄型ケース ＋ 金属製の固定スタンド（鍵・ワイヤー付き） | 見やすい角度に固定でき、持ち去り・転倒を防止。金属なので拭ける | スタンド 約0.8〜1.5万円 |
| **個室（手渡し）** | 防滴・耐衝撃ケース（持ち手＋肩掛けベルト付き） | 運びやすく落下に強い。防滴で拭ける | 約0.3〜0.6万円 |

代表製品の例：固定スタンド＝エレコム ESL-ST04 系／防滴ケース＝サンワサプライ PDA-TABWPST10BK（IP47）。

## 4. 注文画面の固定（キオスク化）

注文アプリの画面だけを表示し、他のアプリや設定を触れないように固定します。

- **方式：** Fully Kiosk Browser（買い切り **1台あたり約1,400円**、月額不要）
- 夜間は自動で画面オフ→朝に自動起動、トラブル時は自動復旧する運用が可能です。
- 月額制の管理ソフトは不要のため、ランニングコストはほぼかかりません。

## 5. 充電・保管・通信（16台分の共通設備）

| 項目 | おすすめ | 価格目安 |
| :--- | :--- | :--- |
| 充電（まとめて） | 多ポートUSB充電器 ×2 | 約4万円 |
| 保管（施錠） | 鍵付きタブレット保管庫 | 約10〜17万円 |
| 店内Wi-Fi | 業務用Wi-Fiルーター（家庭用より安定） | 約2.4〜3万円（1台で18台分OK） |
| 予備機 | 故障差し替え用に1〜2台 | 本体価格×1〜2 |

> **重要：店内Wi-Fiは「つなぐ台数」より「動画を流す通信量」が混雑の原因**になります。スクリーンセーバーの動画は**端末本体に保存して再生**する設計にすれば、Wi-Fi・インターネット回線への負荷をほぼゼロにできます（開発側で対応します）。

---

## 6. ご注意・運用のポイント

- **常時給電の安全対策：** 電源つなぎっぱなしはバッテリーが膨らむ原因になります。充電を80%で止める設定・夜間のみ充電するタイマー運用で対策します。
- **通信が切れても注文を失わない仕組み：** Wi-Fiが一瞬切れても注文が消えないよう、開発側で「一時保存→復旧時に送信」を実装します。
- **衛生面：** ケースは拭ける素材に統一し、アルコール清掃ができる構成にします。

---

## 7. 次のステップ（おすすめの進め方）

1. **まず推奨機種（アイリス LUCA）を1台だけ先行購入**し、店内の明るさでの**画面の見やすさ（特に解像度）**・ケースの装着感・充電の干渉・画面固定の動作を実機で確認。
2. 問題がなければ**16台をまとめて発注**（同一ロットで揃えると故障対応が楽）。量販店の法人窓口やアイリスオーヤマ法人ルートでまとめ買いすると割引が期待できます。

この方法が、無駄な買い直しのない最も確実な進め方です。
